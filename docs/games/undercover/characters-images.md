# Undercover — Liste des images à sourcer

**95 personnages** à illustrer. Format cible recommandé :
- **Dimensions** : carré 400x400 ou 500x500
- **Format** : `.png` (fond transparent idéal) ou `.jpg`
- **Emplacement final** : `src/assets/img/undercover/<id>.png`
- **Poids** : viser < 150KB / image (compresser avec [tinypng.com](https://tinypng.com) ou [squoosh.app](https://squoosh.app))

Conseil : cliquer directement sur le lien **Google Images** pour chaque perso, filtrer en "Taille : carré" ou "Taille : grand".
